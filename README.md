# Qdumpfs

qdumpfs is a modified version of pdumpfs.

## Installation

gem install qdumpfs

```ruby
gem 'qdumpfs'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install qdumpfs

## Usage

qdumpfs srcdir dstdir
